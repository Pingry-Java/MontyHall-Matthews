OVERVIEW
========
This program iterates a Monty Hall program 10 million times, and returns the probability
of each choice, switching or staying

USAGE
=====
Compile in the MH/ subdirectory with javac Monty_Hall
run java Monty_Hall
to access simulation mode, give the number of iterations wanted in the arguments
in the same line as java Monty_Hall.
