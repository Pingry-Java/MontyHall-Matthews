OVERVIEW
========
This program iterates a Monty Hall program 10 million times, and returns the probability
of each choice, switching or staying

USAGE
=====
Compile rand with javac rand.java, while in the rand file.
Compile Monty_Hall with javac Monty_Hall.java, while in the MH file.
Compile MHI with javac MHI.java, in the parent directory
Run with java MHI, in the parent directory
