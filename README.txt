OVERVIEW
========
This program iterates a Monty Hall program 10 million times, and returns the probability
of each choice, switching or staying

USAGE
=====
The rand.java and Monty_Hall.java programs MUST be in their own subdirectories, within the program directory.
Otherwise, the MHI program will not find the packages necessary to run.

Compile rand with javac rand.java, while in the rand directory.
Compile Monty_Hall with javac Monty_Hall.java, while in the MH directory.
Compile MHI with javac MHI.java, in the parent directory.
Run with java MHI, in the parent directory.
